# Python_Practice

Collection of small Python practice scripts and exercises covering common algorithms,
data-structure manipulations, and utility functions. Useful for interview prep and
learning Pythonic solutions to common problems.

## Contents

- Small scripts (string manipulation, array/list problems, matrix utilities)
- JSON and API response helpers
- Test files and quick experiments

## How to run

1. Create and activate a virtual environment (recommended):

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate
```

2. Run any script with Python, for example:

```bash
python matrix_9_borders.py
```

3. To run tests (if present), use pytest:

```bash
pip install -r requirements.txt  # if you have dependencies
pytest
```

## Notes

- Files are small, self-contained examples. Inspect each file for usage details.
- Add a `requirements.txt` if you install external packages.
